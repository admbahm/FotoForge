# Security Policy

## Supported versions

FotoForge is pre-release. Security fixes are applied to the latest revision on the default branch.

## Reporting a vulnerability

Do not open a public issue for a vulnerability. Use GitHub's private vulnerability reporting for this repository. Include affected versions, reproduction steps, impact, and any proposed mitigation. Do not attach personal media or catalogs containing private metadata.

Maintainers will acknowledge a report within seven days and coordinate disclosure after a fix is available. Safety issues that could cause unintended file movement, overwrite, or deletion are treated as security issues.
